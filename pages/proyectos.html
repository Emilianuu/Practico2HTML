Inteligencia Artificial y Salud: "EcoMind"

EcoMind es una aplicación móvil que utiliza la cámara y sensores del dispositivo para analizar los niveles de estrés y fatiga visual de los programadores en tiempo real. Sugiere pausas activas personalizadas y ajusta automáticamente la temperatura de color de la pantalla según el estado anímico detectado.

Tecnologías usadas:

  • Frontend: Flutter o React Native (para ser multiplataforma).
  • IA/ML: TensorFlow Lite para el procesamiento de imágenes en el dispositivo y OpenCV para la detección de puntos faciales.
  • Backend: FastAPI (Python) para el análisis de datos históricos.

Ciberseguridad y Blockchain: "GhostLedger"

Un sistema de mensajería corporativa donde los mensajes no se guardan en un servidor central, sino que se fragmentan y encriptan en una cadena de bloques privada. Los mensajes se "autodestruyen" después de ser leídos y no dejan rastro forense, ideal para sectores que manejan secretos industriales.

Tecnologías usadas:

  • Red: Hyperledger Fabric o Ethereum (red privada).
  • Contratos Inteligentes: Solidity o Go.
  • Cifrado: Protocolo Signal para el cifrado de extremo a extremo (E2EE) y AES-256.

Desarrollo Web / E-commerce: "StyleMirror"

Un probador virtual de ropa que utiliza Realidad Aumentada (AR) avanzada directamente en el navegador (sin apps externas). Permite a los usuarios "probarse" prendas usando la webcam y ver cómo se mueve la tela según su tipo de cuerpo, integrando pasarelas de pago instantáneas.

Tecnologías usadas:

  • Motor 3D: Three.js o Babylon.js para renderizar la ropa.
  • AR: WebXR Device API o 8th Wall.
  • Backend: Node.js con integración a la API de Shopify o Stripe para los pagos.
